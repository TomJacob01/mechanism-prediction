# Examples

Small runnable scripts demonstrating the package.

| File | What it shows |
| --- | --- |
| [`quickstart.py`](quickstart.py) | Parse → featurize → one forward pass on CPU using only the bundled mock fixture. Smoke-tests a fresh install. |

Run from the repo root:

```bash
python examples/quickstart.py
```

For the full training pipeline see [`scripts/train.py`](../scripts/train.py)
and the project [`README.md`](../README.md).
